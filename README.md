ChatGPT 是 OpenAI 开发的生成式 AI 模型，能用来写内容、做研究、编程、生成图片——用途多到数不过来。

## ChatGPT 的安全措施

ChatGPT 为其 Pro 和 Enterprise 用户提供高级安全措施，包括：

* **数据所有权和保留控制：** 你的公司将能够保留输入和输出的完全所有权，管理员可以设置自定义数据保留期限。
* **军用级加密：** ChatGPT 内的所有数据在静态时使用 AES-256 加密，在传输时使用 TLS 1.2+，符合 SOC 2 Type 1 合规性。
* **细粒度访问控制，**包括用于企业身份验证的基于 SAML 的 SSO、用于限制访问的域验证以及用于功能或数据的基于角色的权限。

![安全功能展示](image/56791862157.webp)

➡️ 尽管如此，第三方集成和自定义配置中存在一些漏洞，包括：

* **配置错误的 GPTs：** 具有外部 API 访问权限的自定义 GPTs 有泄露专有数据或启用钓鱼活动的风险。
* **内存操纵攻击：** 你的工作区可能面临"SpAIware"注入的风险，这些注入将 ChatGPT 变成持久间谍软件，跨会话窃取数据。
* **数据泄露：** 该平台可能通过响应暴露个人信息或关键公司信息，可能会将此类数据用于训练目的。

我还记得 2023 年 4 月的三星 ChatGPT 泄漏事件，当时三星的 3 名员工向 ChatGPT 提供了关键业务信息，如源代码和会议记录以提高生产力——结果却导致公司的知识产权面临风险。

![三星泄漏事件新闻](image/17061323149.webp)

**评分：** 5/10。

尽管 ChatGPT 有合规性和安全控制，但该平台仍然缺乏自主威胁响应和实时网络行为分析，以确保你的数据保持安全。
